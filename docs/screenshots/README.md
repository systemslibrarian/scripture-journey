# Screenshots

Place screenshot images in this directory. The README references:

- `home.png` — Homepage with welcome message and discovery cards
- `lesson.png` — Individual lesson page with Scripture and Continue Your Journey
- `map.png` — Prophecy Map with category groups and progress bars
- `timeline.png` — Timeline view with lessons grouped by biblical era
- `dashboard.png` — Dashboard with progress, quiz stats, and streak
- `quiz.png` — Quiz page with question and feedback states

Recommended: capture at 1280×800 or similar desktop resolution.
