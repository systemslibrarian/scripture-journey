# Scripture Journey

Discover how the whole Bible points to Jesus.

## Stack
- Next.js 14
- TypeScript
- Tailwind CSS

## Run locally

```bash
npm install
npm run dev
```

## Pages
- `/` Home
- `/dashboard` Dashboard
- `/story` Story mode
- `/prophecies` Top prophecies
- `/map` Prophecy map
- `/lessons/the-first-promise` Lesson example

## Notes
This is a starter scaffold for the Scripture Journey web app.
